box-sizing:border-box Consideration

Dialtone has been implemented with the assumption that we are using box-sizing: border-box globally. Please see this Stack Overflow question for an explanation of why we are doing this.

In Vue, we apply box-sizing: border-box globally at the VueView level, ensuring all child elements use this style. As such, Dialtone styles will work correctly in Vue with respect to element sizing.

In Backbone we are not using border-box by default. Because Dialtone expects this, anytime we wish to use Dialtone styles in Backbone we must ensure to apply the box-sizing: border-box style to all affected elements.
