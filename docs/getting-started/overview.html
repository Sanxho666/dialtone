What is Dialtone?

Dialtone is the new design system for Dialpad and UberConference.

What is a design system?

A design system is a collection of reusable components, guided by clear standards, that can be assembled together in a variation of applications. If you're interested learning more about what a design system is, read this article from Audrey Hacq entitled, "Everything you need to know about Design Systems".

Goals

Dialtone has been created with a unified goal to help product and engineering teams create unified, consistent experiences as quickly as possible. To achieve this goal, Dialtone has created a robust set of reusable variables, elements, and components.
