--- layout: page title: Buttons description: Buttons are UI elements which allow users to take actions throughout the app. It is important they are identifiable, consistent, communicate their actions clearly, and are appropriately sized to their action. ---

Styles

Dialtone provides 5 different button styles:

  1. Primary
  2. Secondary
  3. Danger
  4. Muted
  5. Inverted

Primary

The primary button style highlights the most important action(s). To help provide clarity to users, it is recommended that only one button have this style within a section or page.

{% highlight html linenos %} {% endhighlight %}
Name Class Default Selected Disabled
Primary .dt-btn-primary

Secondary

The secondary button styles are provided to help communicate possible, alternate actions the user could take. These are provided in three, graduating styles: clear (default), outlined, and filled. The button style selected is contextually dependent. The more important the action, the heavier the button's visual style should be.

{% highlight html linenos %} {% endhighlight %}
{% for item in site.data.buttons.secondary %} {% assign types = item.classes %} {% for class in types %} {% endfor %} {% endfor %}
Name Class Default Selected Disabled
{{ class.type }} .{{ item.class }} {% if class.class != nil %}.{{ class.class }}{% endif %}
