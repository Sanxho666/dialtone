--- layout: page title: Background Patterns description: Dialtone provides various patterns to be used for Department and Call Centers. Unlike most SVG icons, we are unable to inline these patterns in order to have them repeat vertically correctly. To apply these patterns, Dialtone provides the following background pattern classes. These classes apply the pattern SVGs as base64 background-images. ---
