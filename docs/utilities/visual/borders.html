Border widths

Set a border's width by using a border style utility class. This allows you to not only increase a border width from it's default 1px value, but also remove borders by setting them to 0.

{% highlight html linenos %}
.ba .baw2
.ba .baw2 .btw0
.bt .btw2
.br .brw2
.bb .bbw4
.bl .blw4
.bx .bxw6
.by .byw6
