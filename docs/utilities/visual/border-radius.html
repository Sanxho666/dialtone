--- layout: page title: Border Radius description: Utility classes for changing the border radius of an element. ---

Classes

Class Output Description
.br-sm border-radius: 3px; Adds a 3px border radius.
.br-md border-radius: 5px; Adds a 5px border radius.
.br-lg border-radius: 7px; Adds a 7px border radius.
.br-xl border-radius: 11px; Adds an 11px border radius.
.btr-[size] border-top-left-radius: [size];
border-top-right-radius: [size]; 		Adds a border radius to both top corners.
.brr-[size] border-top-right-radius: [size];
border-bottom-right-radius: [size]; 		Adds a border radius to both right corners.
.bbr-[size] border-bottom-left-radius: [size];
border-bottom-right-radius: [size]; 		Adds a border radius to both bottom corners.
.blr-[size] border-top-left-radius: [size];
border-bottom-left-radius: [size]; 		Adds a border radius to both left corners.
