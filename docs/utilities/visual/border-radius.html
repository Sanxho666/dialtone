--- layout: page title: Border Radius description: Utility classes for changing the border radius of an element. ---

Classes

Class Output Description
.br-sm border-radius: 3px; All corners receive a small border radius (3px).
.br-md border-radius: 5px; All corners receive a medium border radius (5px).
.br-lg border-radius: 7px; All corners receive a large border radius (7px).
.br-xl border-radius: 11px; All corners receive an extra-large border radius (11px).
.btr-[size] border-top-left-radius: [size];
border-top-right-radius: [size]; 		Both top corners (top left, top right) receive a border radius.
.brr-[size] border-top-right-radius: [size];
border-bottom-right-radius: [size]; 		Both right corners (top right, bottom right) receive a border radius.
.bbr-[size] border-bottom-left-radius: [size];
border-bottom-right-radius: [size]; 		Both bottom corners (bottom left, bottom right) receive a border radius.
.blr-[size] border-top-left-radius: [size];
border-bottom-left-radius: [size]; 		Both left corners (top left, bottom left) receive a border radius.
