--- layout: page title: Display description: Change an element's display property by using the following display utility classes. ---

Classes

Class Output Description
.d-block display: block; Gives an element a block-level display value.
.d-inline-block display: inline-block; Gives an element a block-level display value that will flow with surrounding content as if it were a single, inline box (behaving much like a replaced element would).
.d-inline display: inline; Gives an element an inline display value that flows like text.
.d-flex display: flex; Gives an element a flex display value that will lay out the element's immediate children in a non-wrapping row.
.d-inline-flex display: inline-flex; Gives an element an inline flex display value, behaving like an inline element but adopting many of the flex model attributes.
.d-grid display: grid; Gives an element a grid display value.
.d-inline-grid display: inline-grid; Gives an element an inline grid display value.
.d-table display: table; Gives an element a table display value, having it visibly display like an HTML table element. It defines a block-level element.
.d-table-cell display: table-cell; Makes an element behave like an HTML table cell (td) element.
.d-none display: none; Makes an element behave like an HTML table cell (td) element.
.d-initial display: initial; Makes an element behave like an HTML table cell (td) element.
