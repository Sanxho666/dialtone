Classes

Class Output Description
.dt-of-auto overflow: auto; Automatically determines how overflow should respond to overflowing content.
.dt-of-x-auto overflow-x: auto; Automatically when to show scrollbars on the element's x-axis.
.dt-of-y-auto overflow-y: auto; Automatically when to show scrollbars on the element's y-axis.
.dt-of-hidden overflow: hidden; Hides an element's overflowing content.
.dt-of-x-hidden overflow-x: hidden; Hides an element's overflowing content along the x-axis.
.dt-of-y-hidden overflow-y: hidden; Hides an element's overflowing content along the y-axis.
.dt-of-scroll overflow: scroll; Scrolls an element's overflowing content.
.dt-of-x-scroll overflow-x: scroll; Scrolls an element's overflowing content along the x-axis.
.dt-of-y-scroll overflow-y: scroll; Scrolls an element's overflowing content along the y-axis.
.dt-of-visible overflow: visible; Makes an element's overflowing content visible (default).
