Auto

Using .dt-of-auto, .dt-of-x-auto, or .dt-of-y-auto will add autobars automatically when an element's content exceeds the element's visual size. The .dt-of-x-auto and .dt-of-y-auto classes will only show scrollbars vertically or horizontally, respectively.
