--- layout: page title: Stacks description: Stacks are preset layouts which inject margins between sibling elements within a parent element. This method is favored over applying margins to an element, as it allows for greater modularity and flexibility with layouts. ---

The stack layout works by using the adjacent sibling combinator (+) to apply a top margin when an element is preceded by another element. Furthermore, to allow for nesting stacks and controlling the cascade, these margins are scoped to only apply to children of the parent wrapper (e.g. .stack-layout > * + *).
