--- layout: page title: Vertical Alignment description: Utility classes for controlling an object's vertical alignment inline or table-cell box. ---

Classes

Class Output Description
.va-baseline vertical-align: baseline; Aligns the element's baseline with the parent's baseline.
.va-bottom vertical-align: bottom; Aligns the bottom of the element and its children with bottom of the entire line.
.va-middle vertical-align: middle; Aligns the middle of the element with the baseline plus half the x-height of the parent.
.va-sub vertical-align: sub; Aligns the baseline of the element with the subscript baseline of the parent.
.va-super vertical-align: super; Aligns the baseline of the element with the superscript baseline of the parent.
.va-text-bottom vertical-align: text-bottom; Aligns the bottom of the element with the bottom of the parent's element font.
.va-text-top vertical-align: text-top; Aligns the top of the element with the top of the parent's element font.
.va-top vertical-align: top; Aligns the top of the element with the top of the entire line.
.va-unset vertical-align: unset; Resets vertical alignment to the element's default value.
