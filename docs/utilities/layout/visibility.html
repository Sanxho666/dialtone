--- layout: page title: Visibility description: A CSS property that shows or hides an element without changing the layout of a document. It can also be used to hide rows or columns in tables. ---

Classes

Class Output Description
.dt-v-visible visibility: visible; Makes an element visible.
.dt-v-hidden visibility: hidden; Hides an element visually, but leaves it in the layout.
.dt-v-collapse visibility: collapse; For table rows, row groups, columns or column groups; the space the element would have occupied is removed (as if display: none; were applied.) Flex items are hidden and the space it would have occupied is removed. For all other elements, it's treated as hidden.
.dt-v-initial visibility: initial; Resets visibility to the initial value of visibility, which is visible.
