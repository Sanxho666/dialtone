--- layout: page title: Overflow description: A CSS property that tells an element what to do when its content is too big to fit its block-level context. ---

Classes

Class Output Description
.of-auto overflow: auto; Automatically determines how overflow should respond to overflowing content.
.of-x-auto overflow-x: auto; Automatically when to show scrollbars on the element's x-axis.
.of-y-auto overflow-y: auto; Automatically when to show scrollbars on the element's y-axis.
.of-hidden overflow: hidden; Hides an element's overflowing content.
.of-x-hidden overflow-x: hidden; Hides an element's overflowing content along the x-axis.
.of-y-hidden overflow-y: hidden; Hides an element's overflowing content along the y-axis.
.of-scroll overflow: scroll; Scrolls an element's overflowing content.
.of-x-scroll overflow-x: scroll; Scrolls an element's overflowing content along the x-axis.
.of-y-scroll overflow-y: scroll; Scrolls an element's overflowing content along the y-axis.
.of-visible overflow: visible; Makes an element's overflowing content visible (default).
