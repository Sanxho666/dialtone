--- layout: page title: Position description: A CSS property that sets how an element is positioned within the document. ---

Classes

Class Output Description
.ps-absolute position: absolute; Removes the element from the normal document flow and absolutely positions relatively to its closest positioned ancestor. Final positioning is used using top, bottom, left, or right properties.
.ps-fixed position: fixed; Removes the element completely from the normal document flow and fixes its position relative to the viewport. Final positioning is used using top, bottom, left, or right properties.
.ps-relative position: relative; Positions an element within the normal document flow, and then offsets relative to itself based on top, bottom, left, or right values.
.ps-static position: static; Positions an element within the normal document flow. The top, bottom, left, right, and z-index properties have no affect. This is the default value.
.ps-sticky position: sticky; Positions an element within the normal document flow, and then offset relative to its nearest scrolling ancestor and containing block. This creates a new stacking context.
.ps-initial position: initial; Resets an element's position to the initial value (static).
.ps-unset position: unset; Unsets a element's position, restoring it's initial value (static).
