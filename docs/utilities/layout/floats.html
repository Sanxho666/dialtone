float

Created primarily as a way to float images within copy blocks, the float property allows you to remove an object out of the document flow and place it elsewhere within the document. While floating an object gives you more layout abilities, it also removes it from the document flow and so the parent loses it's ability to make sure it correctly encapsulates it. To fix this you must use the "clearfix" hack.

Classes

Class Output Description
.fl-left float: left; Floats an element to the left.
.fl-right float: right; Floats an element to the right.
.fl-none float: none; Removes the ability for the element to float.
.fl-initial float: initial; Restores an element's initial float value (unset).
.fl-unset float: unset; Resets the elements float value to its initial value.

    clear: both;
    content: "";

clear

The clear CSS property allows an object to escape floated objects on the left, right, or both sides. This is used to make sure floated objects don't float on top of each other or create odd layouts.

Classes

Class Output Description
.cl-left clear: left; An element is moved down to clear past left floats.
.cl-right clear: right; An element is moved down to clear past right floats.
.cl-both clear: both; An element is moved down to clear past left and right floats.
.cl-none clear: none; An element is not moved down to clear past floating elements.
.cl-initial clear: initial; Restores an element's initial cleared float value.
.cl-unset clear: unset; Unsets any cleared float values.
