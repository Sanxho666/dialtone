--- layout: page title: Floats description: A CSS property that allows copy to flow around an element, typically an image. ---

Classes

Class Output Description
.fl-inline-start float: inline-start; An element must float on the start side of its containing block. That is the left side with left-to-right (LTR) scripts and the right side with right-to-left (RTL) scripts.
.fl-inline-end float: inline-end; An element must float on the end side of its containing block. That is the left side with LTR scripts and the right side with RTL scripts.
.fl-left float: left; Floats an element to the left.
.fl-right float: right; Floats an element to the right.
.fl-none float: none; Removes the ability for the element to float.
.fl-initial float: initial; Restores an element's initial float value (unset).
.fl-unset float: unset; Resets the elements float value to its initial value.
.cl-inline-start clear: inline-start; An element is moved down to clear floats on the start side of its containing block. That is the left side with left-to-right (LTR) scripts and the right side with right-to-left (RTL) scripts.
.cl-inline-end clear: inline-end; An element is moved down to clear floats on the end side of its containing block. That is the left side with LTR scripts and the right side with RTL scripts.
.cl-left clear: left; An element is moved down to clear past left floats.
.cl-right clear: right; An element is moved down to clear past right floats.
.cl-both clear: both; An element is moved down to clear past left and right floats.
.cl-none clear: none; An element is not moved down to clear past floating elements.
.cl-initial clear: initial; Restores an element's initial cleared float value.
.cl-unset clear: unset; Unsets any cleared float values.
.cl-fix .cl-fix:before, .cl-fix:after { content: ""; clear: both; display: table; } A CSS hack for when all an element's children are floated and the parent's height isn't correctly calculated. This helps visually restore an elements height.
