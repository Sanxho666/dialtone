Auto Columns

The grid-template-auto CSS property defines how items which fall outside of the grid (i.e. the implicit grid) should lay out. For example, say we have 2 defined columns (explicit grid) and we want to position an element in a third column. To do this we would state our explicit grid (.gtc-value) and our .gac-value class, passing the following CSS variable: --grid-auto-col: 320px;. We would also assign the column to our element using the .ga-value class, passing the following CSS variable: --grid-area: 1 / 3 / 2;
