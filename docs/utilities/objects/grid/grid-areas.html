Grid Areas

As you create a grid, you can specify where items should be placed in the grid. This can be done two ways. First, you can assign names to grid cells. This is done using the grid-template-areas CSS property. For example, we can create a grid with a sidebar and content areas by using the .dt-gta-value and passing the following CSS variable: --grid-template-area: 'sidebar content';. To assign child elements to these template areas, use the .dt-ga-value class (grid-area).

Sometimes though you might not be able to assign grid areas. In these cases you can still declare how an element should lay out in the grid by telling it what line number it should start, stop, or span. For example, to have an element start on the 2 cell and span 2 cells, we would use the .dt-ga-value class, passing the following CSS var: --grid-area: 2 / span 2;.
