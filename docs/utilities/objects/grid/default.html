Default grid

The default grid is initiated with the .dt-grid class. This starts a two-dimensional CSS grid. The number of columns within the grid are determined by the number of direct children. Grid cells widths have a default minmax() value of var(--grid-col-min-width) (5rem) and max-content respectively. You can change the minimum value by updating the min-width variable inline.
