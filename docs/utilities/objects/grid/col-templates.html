Template Columms

The grid-template-columns CSS property defines the column line names and track sizes. For example, say we want to create 3 columns, the first 208px, the third one 120px, and the middle column to take up all the remaining space. We would do this by using .d-gtc-value class and and passing the following CSS variable: --grid-template-col: 13rem 1fr 7.5rem;.
