Auto Rows

The grid-auto-rows CSS property defines how rows that fall outside of the implicit grid (i.e. grid-template-rows) should lay out. For example, say we have 2 by 2 grid and we want to position an element in a third row. To do this we need to set a grid-auto-rows value with the .dt-gar-value class, modifying the following CSS variable if needed: --grid-auto-row. We also will need to assign an element to the row using the .dt-ga-value class.
