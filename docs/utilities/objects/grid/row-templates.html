Template Rows

The grid-template-rows CSS property defines the row line names and track sizes. For example, say we want to create 4 rows, the first row is 80px tall, the second row is 160px tall, the last row is 240px, and the third column we want to be at least 240px but take up any remaining space if more exists. We would do this by using .gtr-value class and and passing the following CSS variable: --grid-template-row: 5rem minmax(15rem, 1fr) 2.5rem;.
