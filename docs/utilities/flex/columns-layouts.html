{% header "h2", "Flex Columns" %}

By default children cells within a parent flex container will only take up the room needed for their content. To have a flex child cell span multiple columns, use the flex column .d-flex__cell{#} classes. These columns are based on a 12-column grid, meaning each column will take up 1/12th of all available space.

{% header "h2", "Centering objects" %}

By default flexed items align to flex-start both horizontally and vertically (effectively top, left). This aligns child elements center-center. Note: This only applies for direct children and does not cascade to subsequent children.
