{% header "h2", "Space Around" %}

Use {% code %}d-jc-space-around{% endcode %} to justify items along the element's main axis so that there is an equal amount of space on each side of the item. This effectively takes all available space, divides it for each child element, placing half of available space on either side of the child. This is why the space appears doubled for interior objects versus end objects.
