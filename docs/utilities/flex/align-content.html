{% header "h2", "Space Around" %} {% paragraph %}Use {% code %}d-ac-space-around{% endcode %} to pack rows along the element's cross axis so that there is an equal amount of space on each side of the item. This effectively takes all available space, divides it for each row, placing half of alotted space on either side of the row. This is why the space appears doubled for interior rows versus end rows.{% endparagraph %}
