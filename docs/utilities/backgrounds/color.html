{% header "h2", "Changing opacity" %}

Use d-bgo{stop} to change an element's background color opacity. You can also change the background color opacity on :hover, :focus, or in dark mode by using the respective h:d-bgo{stop}, f:d-bgo{stop}, or d:d-bgo{stop} prefix.
