Variables

When writing Less, you can set an element's text color by using a @line-height--{size} variable (ex. p {line-height: @line-height--md;}). Useful when writing scoped Less where an immutable class with !important isn't required.

{% for i in type.line-height %} {% assign class = i.class %} {% assign output = i.output %} {% endfor %}
Variable Output
@line-height--{{ class }} {{ output }}

Classes

Classes are immutable, meaning they include !important to override CSS specificity. Useful when styling HTML by including CSS classes in markup.

{% for i in type.line-height %} {% assign class = i.class %} {% assign output = i.output %} {% endfor %}
Class Output
.d-line-height-{{ class }} line-height: {{ output }} !important;
