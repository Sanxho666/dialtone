Font Family

Variables

When writing Less, you can set an element's text color by using a @font-family--{family} variable (ex. p {font-family: @font-family--sans;}). Useful when writing scoped Less where an immutable class with !important isn't required.

{% for i in type.family %} {% assign class = i.class %} {% assign output = i.output %} {% endfor %}
Variable Output
@font-family--{{ class }} {{ output }}

Classes

Classes are immutable, meaning they include !important to override CSS specificity. Useful when styling HTML by including CSS classes in markup.

{% for i in type.family %} {% assign class = i.class %} {% assign output = i.output %} {% endfor %}
Class Output
.d-font-family-{{ class }} font-family: {{ output }} !important;

Examples

{% for i in type.family %} {% assign class = i.class %}

.d-font-family-{{ class }}
The weather had changed overnight, when a backing wind brought a granite sky and a mizzling rain with it.

{% endfor %}
