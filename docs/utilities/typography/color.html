--- layout: page title: Colors description: Utilities to change an element's font-color. ---

All font colors pass the WCAG 2.1 Level AA contrast ratio requirements (ratio >= 4.5:1) and most pass WCAG 2.1 Level AAA requirements (ratio >= 7:1). The contrast ratio value is noted with the colors below. Please use only these colors or variations of these colors which pass WCAG 2.1 Level AA contrast ratio requirments.

{% header "h2", "Changing opacity" %}

Use d-fco{n} to change an element's text color opacity. You can also change font color opacity on :hover, :focus, or in dark mode by using the respective h:d-fco{stop}, f:d-fco{stop}, or d:d-fco{stop} prefixes.
