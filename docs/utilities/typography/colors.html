--- layout: page title: Colors description: Dialtone provides a limited set of text-color utility variables and classes. ---

Along with these colors, we've provided text and background color pairings. Most of these pairings pass the WCAG 2.1 Level AAA contrast ratios; however some only pass WCAG 2.1 Level AA. These are noted with the colors below. Please use only these colors. Using these colors helps ensure we are meeting web accessibility standards.

Variables

When writing Less, you can set an element's text color by using a @color--{color} variable. The table below lists the available variables and its output.

Variable Output

Classes

Grays

All grays meet the minimum contrast ratio for WCAG 2.1 Level AAA, which is 7:1.
