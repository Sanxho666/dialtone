Variables

When writing Less, you can set an element's font-size by using a @{headline/text}--{size} variable. The table below lists the font-size variables, size, and their output.

{% for i in site.data.type.font-size %} {% assign class = i.class %} {% assign px = i.pixel %} {% assign output = i.output %} {% endfor %}
Variable Size Output
@{{ class }} {{ px }}px {{ output }}rem

Classes

Set an element's font-size by using the .d-{headline/text}--{size} classes. The table below lists the available classes, size, and their CSS output.

{% for i in site.data.type.font-size %} {% assign class = i.class %} {% assign px = i.pixel %} {% assign output = i.output %} {% endfor %}
Variable Size Output
.d-{{ class }} {{ px }}px {% if class contains "headline" %} font-size: {{ output }}rem;
font-weight: 600; {% if class == "headline--xs" %}
text-transform: uppercase; {% endif %} {% elsif class contains "text" %} font-size: {{ output }}rem; {% endif %}
