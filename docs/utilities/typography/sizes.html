Variables

When writing Less, you can set an element's text color by using a @{headline/text}--{size} variable (ex. p {font-size: @text--md;}). Useful when writing scoped Less where an immutable class with !important isn't required.

{% for i in type.font-size %} {% assign variable = i.variable %} {% assign px = i.pixel %} {% assign output = i.output %} {% endfor %}
Variable Size Output
@{{ variable }} {{ px }}px {{ output }}rem

Classes

Classes are immutable, meaning they include !important to override CSS specificity. Useful when styling HTML by including CSS classes in markup.

{% for i in type.font-size %} {% assign class = i.class %} {% assign px = i.pixel %} {% assign output = i.output %} {% endfor %}
Variable Size Output
.d-{{ class }} {{ px }}px {% if class contains "headline" %} font-size: {{ output }}rem; !important
font-weight: 600 !important; {% if class == "headline-xs" %}
text-transform: uppercase; !important {% endif %} {% elsif class contains "text" %} font-size: {{ output }}rem; !important {% endif %}

Examples

{% for i in type.font-size %} {% assign class = i.class %}
.d-{{ class }}

The quick brown fox jumps over the lazy dog.
